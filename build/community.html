Community

Welcome to the Cabin community! By being here, you're already one of us! We're called campers, and we're delighted to have you. Currently, the Cabin community is very small, but we hope to grow into a large and diverse group.

Cabin has communities in a few places. Most general discussion and community building happens in the Cabin Discord server.

Discord

https://discord.gg/ZcfC322pAb

The Cabin Discord server is the best place for interacting with the community! Feel free to stop by and chat! The Discord server has channels for help with the language, general discussion, suggestions, updates, and more. It's a welcoming and friendly place for all campers!

GitHub

https://github.com/cabin-language

The Cabin GitHub organization holds all of the source code behind the Cabin as a project. Even the code behind this very website is there! Specifically, the following projects are all open source:

  • The Cabin API
  • The Cabin compiler executable
  • The Cabin Tree-Sitter parser
  • The Cabin website
  • The Cabin Neovim plugin
  • The Cabin Visual Studio Code plugin

The GitHub is a good place if you want to:

  • File an issue or bug report
  • Discuss potential changes or new features to the language
  • View the source code for Cabin projects

Help Wanted!

Cabin is looking for contributors! There are a number of ways you can contribute to the project; Specifically, we're looking for:

  • A logo
  • Bug reports
  • Documentation
  • Language Suggestions

...and anything else you have to offer! If you're interested in contributing, you can start immediately, depending on how you want to contribute. Bug reports can be filed immediately on the Cabin GitHub. Documentation contributions are welcome as pull requests in the Cabin GitHub as well. For branding contributions, such as creating a logo for Cabin, get in touch with me, Cabin's creator, Vi. You can find me in the Cabin Discord.
