Documentation

Here you'll find an index of documentation for the Cabin programming language. If you're not sure where to start, it's recommended to start with The Cabin tutorial .

The Cabin Tutorial

The Cabin tutorial should be your first starting point with Cabin. It'll walk you through what the language is, how to get it installed on your system and set up in your editor, and how to get started programming with Cabin, going over all of the language's features and syntax, as well as important built-ins and libraries to be aware of.

The Cabin Reference

The Cabin reference page provides exhaustive overviews of all aspects of the language, including syntax, builtins, compiler usage, and more. It's a reference document meant for quick lookups and information, not learning the language or end-to-end reading. For general language usage, it's recommended to view The Cabin tutorial instead.

Compiler Command Reference

The compiler command reference is a list of all options, arguments, and flags that can be passed to the Cabin compiler, with comprehensive descriptions for what they all do and when to use them.

Standard Library Reference

The standard library reference is a fully exhaustive reference for every exposed binding built into the language.

Language Syntax Reference

The language syntax reference contains various representations of Cabin's grammar, as well as lists of keywords, reserved identifiers, etc.
