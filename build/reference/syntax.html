Keyword Reference

Cabin only has 15 keywords. Here you'll find a complete list of them, as well as a brief description of their usage and a link to the documentation that explains them.

KeywordDocumentationDescription
actionActionsDefines a callable sequence of statements.
andTODOReturns whether both boolean operands are true , lazily evaluating its operands and short-circuiting if necessary.
eitherEithersCreates a tagged union.
extensionofExtensionsExtends a type by adding additional properties onto it.
ifIf ExpressionsConditionally runs code if a boolean condition is true.
isControl FlowJumps to a label, setting it to the given value.
foreachLoopsRuns a block of code for each element in a list.
groupGroupsCreates an instantiable type representing a group of values.
letDeclarationsBinds a name to a value
notTODONegates a boolean value.
orTODOReturns whether either of the boolean operands are true , lazily evaluating its operands and short-circuiting if necessary.
otherwiseIf ExpressionsRuns a block of code conditionally if a condition is not true.
runRuntime EvaluationForces an expression to be evaluated at runtime.
tobeExtensionsExtends one type to be another.
whileLoopsRuns a block of code while a boolean condition is true.
