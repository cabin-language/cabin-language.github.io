Text

Text is Cabin's equivalent to String in most languages. It represents... well, text. Text is a group, but it's a special group that can't be instantiated with the standard new syntax; Instead, quoted text literals are used to instantiate it:

Properties

bytes: action(this: This): Number

Gets the length of the string, in bytes.

ends_with: action(this: This, pattern: This): Boolean

Returns whether this text ends with the given suffix.

length: action(this: This): Number

Gets the length of the string, in graphemes.

lines: action(this: This): List<This>

Splits this text into lines.

replace: action(this: This, pattern: This, replacement: This): List<This>

Replaces substrings with a given replacement.

split: action(this: This): List<This>

Splits this text by whitespace.

split_any: action(this: This, delimiters: List<This>): List<This>

Splits this text by the given delimiters.

starts_with: action(this: This, pattern: This): Boolean

Returns whether this text starts with the given prefix.

trim: action(this: This): This

Removes whitespace from the beginning and end of this text.

trim_end: action(this: This): This

Removes whitespace from the end of this text.

trim_start: action(this: This): This

Removes whitespace from the beginning of this text.
