Cabin Reference

On this page you'll find a few quick references relating to the Cabin language. These are more like records and data than pages worth reading through in their entirety. For learning the language, it's recommended to read through the tutorial instead.

Compiler Command Reference

The compiler command reference is a list of all options, arguments, and flags that can be passed to the Cabin compiler, with comprehensive descriptions for what they all do and when to use them.

Standard Library Reference

The standard library reference is a fully exhaustive reference for every exposed binding built into the language.

Language Syntax Reference

The language syntax reference contains various representations of Cabin's grammar, as well as lists of keywords, reserved identifiers, etc.
