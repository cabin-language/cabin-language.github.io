Introduction

Welcome to Cabin! This documentation is designed for newcomers to the language. It'll walk you through the basics of Cabin and how to get started with the language, all the way from your first "Hello world" program to complex programs with external libraries.

What is Cabin?

Cabin is a programming language designed to be safe , simple , and speedy . To be specific, it's a memory-safe, statically typed, compiled, multi-paradigm programming language.

Cabin has a number of features that makes it unique, but most notably is it's treatment of compile-time evaluation . Cabin allows arbitrary expressions to be evaluated at compile-time, similar to other modern languages such as Zig and Jai. However, Cabin takes this a step further by always evaluating expressions at compile-time by default if it is able to be. This leads to a superb developer experience where complex bugs can be caught at compile-time and displayed in real-time in an editor.
